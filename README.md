# 🏡 Staycasa

Staycasa is a full-stack web application designed for users looking for **short-term, month-based stays** for vacations and temporary living.  
It connects users with property owners, making it easy to discover, book, and manage stays with a smooth and modern experience.

---
## 🚀 Live Demo

- 🌐 Frontend: https://www.staycasa.co.in  
- 🔗 Backend API: https://api.staycasa.co.in  

---

## 🎯 Project Goal

Staycasa helps users find and book **short-term and monthly rental stays** for vacations or temporary relocation, while enabling property owners to list and manage their properties efficiently.

---

## ✨ Features

### 👤 User Features
- User registration & login (JWT + HTTP-only cookies)
- Google OAuth authentication
- Browse and explore available properties
- Book short-term/month-based stays
- View and manage booking history
- Wallet system integration
- Chat with property owners
- Rate and review properties

---

### 🏠 Owner Features
- Owner registration & login
- Owner dashboard
- Property management (add/edit/delete listings)
- Booking management
- Subscription plan system
- Chat with users

---

### 🛠️ Admin Features
- Admin login & dashboard
- User management
- Owner management
- Property management
- Booking management
- Subscription management
